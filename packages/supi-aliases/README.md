# @mrclrchtr/supi-aliases

Command aliases for the [pi coding agent](https://github.com/mariozechner/pi-coding-agent).

## Install

```bash
pi install npm:@mrclrchtr/supi-aliases
```

## What it adds

This extension registers three commands:

- `/exit` — quit pi
- `/e` — shorthand for `/exit`
- `/clear` — start a new session, matching `/new`

## Usage

```text
/exit
/e
/clear
```

## Configuration

This package has no runtime configuration.

## Requirements

- `@mariozechner/pi-coding-agent`
