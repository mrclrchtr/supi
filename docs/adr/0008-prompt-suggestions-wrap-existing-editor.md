# Prompt suggestions wrap the existing editor

`@mrclrchtr/supi-prompt-suggestions` will wrap PI's currently configured editor component instead of assuming it owns the editor outright. Directly replacing the editor would be simpler, but PI extensions can already provide custom editors; wrapping preserves composition with modal editors, border/status editors, and future SuPi editor customizations while keeping prompt suggestions focused on rendering and accepting advisory text. When PI is using its built-in default editor, `ctx.ui.getEditorComponent()` returns `undefined`; the prompt-suggestions wrapper will use `CustomEditor` as its default-editor fallback rather than treating that as "no editor exists". Ghost text is accepted via Right Arrow (inserts text without submitting). Escape and all other keys pass through to the inner editor normally; typing any character dismisses the ghost suggestion before the character is processed. Ghost text is rendered using PI's `"dim"` theme color for visual distinction from entered text.
